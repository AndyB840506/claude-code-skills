# Glosario Cachaco — Lenguaje Bogotano de los 40's

## Términos frecuentes para los guiones

| Término | Significado | Uso en contexto |
|---|---|---|
| **Ala** | Expresión de sorpresa, asombro | "¡Ala! ¿Ese riff es de qué año?" |
| **Chirriado** | Elegante, fino, bien hecho, hermoso | "Ese análisis quedó chirriado, sumerce" |
| **Pútrido** | Excelente, genial, máximo nivel (en este contexto positivo) | "Eso estuvo pútrido, hermano" |
| **Cachifo** | Joven, muchacho, artista emergente | "Los cachifos de hoy están tocando en La Cantera" |
| **Sumerce** | Usted (tratamiento formal/respetuoso cachaco) | "¿Qué opina sumerce sobre eso?" |
| **Caray** | Expresión de énfasis, sorpresa | "¡Caray, qué riff!" |
| **Carachas** | Expresión de sorpresa/énfasis (variante) | "¡Carachas, eso fue brutal!" |
| **A la orden** | Disponible, listo, presente | "Estoy a la orden para el concierto" |
| **Divino** | Excelente, perfecto, hermoso | "La presentación quedó divina" |
| **Soberano** | Máximo, supremo, de lo mejor | "Ese fue un soberano concierto" |
| **Qué pena** | Qué lástima, qué pesar | "Qué pena que no pudiste ir" |
| **De malas** | De mala suerte, desafortunado | "El show se pospuso, quedamos de malas" |
| **Bacano** | ❌ NO USAR (muy moderno, no cachaco) | — |
| **Parcero** | ❌ NO USAR (muy moderno, no cachaco) | — |
| **Vosotros** | ❌ NO USAR (España) | — |
| **Vos** | ❌ NO USAR (paisa/argentino) | — |

## Estructuras de frase frecuentes

**Apertura de conversación:**
- "¿Qué le parece, sumerce, si empezamos con...?"
- "A la orden, vamos a hablar de..."
- "Caray, esta historia vale la pena contar"

**Transiciones entre temas:**
- "Bien chirriado lo que acabamos de escuchar, ahora..."
- "Ala, y hablando de eso, los cachifos de hoy están..."
- "Soberano el momento, pero déjame contarle..."

**Cierre/reflexión:**
- "Divino lo que descubrimos hoy"
- "Eso quedó pútrido, hermano"
- "A la orden para el próximo episodio"

## Notas importantes

- **NO mezclar lenguaje moderno con cachaco** — evita "bacano" o "parcero" aunque suene natural
- **Usted/Sumerce consistentes** — en los guiones se tutea entre Andrés y Juan, pero al oyente se usa "sumerce" o "usted"
- **Tono:** formal pero cercano, elegante sin ser afectado
- **Contexto:** este es el lenguaje de Bogotá de los 40's, no de otras regiones colombianas ni otros países
