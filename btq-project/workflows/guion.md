# Workflow: Guión de Episodio BTQ

> **Router — El workflow completo vive en `/btq-guion`.**

Para crear o revisar guiones de episodios BTQ, invocar directamente:

```
/btq-guion
```

O escribir cualquier trigger: "guión BTQ", "script episodio", "escribir guión EP.0XX".

---

**¿Por qué aquí?** `btq-project` gestiona todo el proyecto BTQ excepto guiones. Los guiones tienen su propia skill dedicada (`btq-guion`) con el workflow completo de 8 pasos, Tone Master, formato narrativo, checklist de calidad y entrega a Google Drive.

**No dupliques instrucciones aquí.** La fuente canónica es `btq-guion/SKILL.md`.
