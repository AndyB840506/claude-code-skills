# Quote Cards — Case File No. 001: Eight Is Great

Static images (no animation — token/iteration cost ruled that out, per the same decision
BTQ/MPD already made). One quote per card, square 1:1, CCC's own visual system: ink black
`#141414` background, aged parchment `#E8DCC4` for quote text, crimson `#8B1A2B` for
accents/attribution — same palette as `artwork-general.md` and `artwork-ep001.md`.

**Attribution note (read before generating):** three of these six lines come from the
script's *dramatized* first-person sections (the composite banker, and Carrie Tolstedt's
reconstructed internal monologue voicing her documented conduct) — not verbatim quotes
either person is on record saying. Those are attributed to the show, not to a named real
person, to avoid misrepresenting dramatization as a direct quote. Only Card 3 (John
Stumpf) is a real, independently sourced quote and is attributed to him by name — the
script itself flags it `[VERIFICAR]` for exact original wording, so double-check phrasing
against a primary source before this one goes out publicly.

All six lines were verified against the actual recorded episode's transcript — every one
of them was really spoken in the condensed ~23-min cut, not just present in the unrecorded
full script.

## Generation notes

- No image-generation API exists for this — same structural pause as cover art. Generate
  each prompt in Flow/Nani Banana, pick the best candidate, and provide the downloaded
  file paths.
- Unlike the cover art, quote cards don't need the full artwork-validation subagent pass
  (margin rule, split-scene checks) — a quick visual check is enough since the format is
  simpler (one quote, one background, one attribution). Still worth confirming: text
  renders exactly as written, no garbled letters, attribution color/placement consistent
  across all six for a cohesive set.
- Same 500KB-under-Spotify-limit / squoosh.app compression step applies if these get
  uploaded anywhere with a size cap; for Instagram/social posting there's no hard limit
  like Spotify's, so compression is optional there.
