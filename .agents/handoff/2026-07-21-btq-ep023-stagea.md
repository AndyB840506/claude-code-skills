## Notes / Gotchas

- **El workflow de Stage A decía "no generes las imágenes todavía"** — desactualizado
  frente a la práctica real desde EP.021/EP.022. Corregido en `00-roadmap.md` Paso 3:
  ahora Stage A incluye un test de validación local rápido (1024×1024) antes de la pausa
  de grabación; el escalado a producción (3000×3000, 16:9, 9:16, texto compuesto con PIL)
  sigue quedando para más cerca del cierre del episodio.
- **Sesgo de etnia por defecto en Z-Image Turbo:** confirmado 2 veces en esta sesión —
  sin describir rasgos étnicos, el modelo genera por defecto un hombre de rasgos
  asiáticos, sin importar el contexto de la escena. Regla nueva y permanente en
  `comfyui/docs/prompting.md` y memoria (`feedback_comfyui_ethnicity_default_bias.md`):
  cualquier persona genérica (sin referente conocido) debe llevar etnia declarada
  explícitamente en el prompt.
- **Preferencia nueva para portadas pilar SEO:** cuando no hay referente pop, preferir un
  objeto/símbolo literal del contenido del capítulo en vez de una persona genérica —
  patrón confirmado 3/3 (EP.020 diana, EP.022 checklist, EP.023 foco+headset). Documentado
  en `episode-launch/docs/brand-constants.md`.
- **Bug propio de sesión (no de producto):** lanzar ComfyUI vía la tool Bash con un path
  de Windows (`--output-directory E:\AI\outputs`) le comió los backslashes al argumento
  (llegó como `E:AIoutputs`) — hubo que relanzar vía PowerShell. Documentado en
  `comfyui/docs/stack-reference.md` y `troubleshooting.md`: usar PowerShell, no Bash, para
  lanzar ejecutables de Windows con paths que llevan backslash.
- **Otro bug propio:** un JSON de workflow API de ComfyUI escrito a mano quedó con una
  llave de cierre faltante — el server devolvía un 500 genérico sin detalle útil hasta
  revisar `E:\AI\comfyui-stderr.log`, que sí trae el traceback exacto. Documentado en
  `troubleshooting.md`.
- **El servidor ComfyUI quedó corriendo** en el escritorio (`E:\AI`, puerto 8188,
  `argv` con el path correcto) por si Andy quiere seguir iterando manualmente desde el
  navegador (`E:\AI\manual.html`).
- EP.024 no tiene tema definido todavía — bajo el giro a 100% pilar SEO, se investigará
  desde cero cuando arranque esa sesión.
