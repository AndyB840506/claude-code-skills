## Concepto visual

**Tema:** Black Sabbath / Birmingham industrial meets heavy metal darkness  
**Estilo:** 35mm film photography, cinematic, 1970s color grading. No cartoon. No circuit boards. No personas reales.

## Cómo usar en Google Flow

```
──────────────────────────────────────────────────────
  CÓMO USAR ESTOS PROMPTS EN GOOGLE FLOW
──────────────────────────────────────────────────────
  1. Abre: labs.google/fx/tools/image-fx
  2. Selecciona "Imagen 3" o "Nani Banana Pro" (si disponible)
  3. Pega el prompt del formato que necesitas (el bloque
     de texto entre las comillas del código)
  4. Genera — el modelo produce 2-4 variantes
  5. Selecciona la mejor por formato
  6. Descarga
  
  Para la portada principal (formato 1:1):
  7. Convierte PNG a JPEG en squoosh.app
  8. Comprime hasta quedar bajo 500 KB
  9. Sube el mismo archivo a Spotify / Apple Podcasts / Amazon Music
  
  Si el resultado no cumple:
  → Añade al final: "highly detailed, sharp focus, professional photography"
  → Ajusta: "more cinematic fog", "stronger crimson glow from below"
  → Si el texto sale mal: especifica "Helvetica Neue Bold Condensed" como fuente
  → Itera con la misma escena cambiando solo la iluminación o composición
──────────────────────────────────────────────────────
```
