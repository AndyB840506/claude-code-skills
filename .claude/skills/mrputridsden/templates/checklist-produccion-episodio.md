# Checklist de Producción — EP.[NNN] [Título]

## Pre-grabación
- [ ] Guion generado (`scripts/ep[NNN]-guion.html`)
- [ ] Plan de grabación generado
- [ ] `eventos.json` actualizado por Juan (segmento de promo)

## Grabación
- [ ] Episodio grabado
- [ ] Audio editado y exportado

## Post-producción
- [ ] Metadata generada (`ep[NNN]-metadata.md`)
  - [ ] Título SEO
  - [ ] Descripción con contacto (web + emails + redes)
  - [ ] Temporada y número de episodio
- [ ] Artwork del episodio generado (`artwork-ep[NNN].md`)
  - [ ] Prompt Spotify 1:1 (3000×3000px)
  - [ ] Prompt Stories 9:16
  - [ ] Prompt YouTube 16:9
- [ ] Arte para redes generado *(cuando redes estén activas)*
- [ ] Show notes generadas
- [ ] Plan de Social Media generado *(cuando redes estén activas)*
- [ ] HTML exportado (`scripts/ep[NNN]-completo.html`)

## Publicación
- [ ] Subido a Spotify (con metadata completa)
- [ ] Backup git + GitHub
- [ ] Backup Google Drive (si G: está montado)
