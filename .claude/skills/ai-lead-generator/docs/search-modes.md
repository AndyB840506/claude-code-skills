# Search Modes

Three options for finding leads:

## Nativo — WebSearch (Free)

- **Cost:** Free (built-in)
- **Speed:** Normal
- **Coverage:** Global, all industries
- **Best for:** Getting started, no API costs

Limitations: Rate-limited by search provider

## Serper.dev — Google Search API

- **Cost:** Paid (requires API key)
- **Speed:** Fast
- **Coverage:** Real Google results
- **Best for:** Production use, high volume

Features:
- Faster results
- More detailed SERP data
- Better pagination
- Higher reliability

## Híbrido — Serper + Hunter.io

- **Cost:** Paid (requires both API keys)
- **Speed:** Fastest
- **Coverage:** Full + email discovery
- **Best for:** Complete lead intelligence

Features:
- Google search via Serper
- Email finding via Hunter.io
- Contact verification
- Most complete profiles

## Recommended

- **Just exploring?** Use Nativo (free)
- **Small lists (10-50)?** Use Serper
- **Production system (100+)?** Use Híbrido

All modes guarantee real leads — never fabricated data.
