# Episode Model

## Season 2 Structure (EP.010+)

Each episode uses a videogame, film, song, or anime as entry point, then distills principles of experience, leadership, or human design.

### Standard Structure

1. **Hook** — Grab attention with the cultural reference
2. **El puente** — Natural transition to the core topic
3. **Los datos** — Max 1 data point per block with source
4. **Las lecciones** — Anchored in the cultural reference
5. **Referencias cruzadas** — Link to other media or past episodes
6. **Aplicable Hoy** — 3 concrete actions the listener can take
7. **Mito o Realidad** — Minimum 3 rounds of myth-busting
8. **Recomendaciones de Andy** — Personal recommendations
9. **Cierre con anécdota** — Single, third-person story
10. **Teaser EP siguiente** — Tease next episode + Tone Master close

### Guidelines

- One concept per episode
- Treat the cultural reference as the thesis, not just decoration
- Data supports the narrative, not the reverse
- Make it relatable to 35–42 year old audience (Gen X references resonate)
