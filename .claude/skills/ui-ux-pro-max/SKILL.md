# UI/UX Pro Max — Design Intelligence

Searchable database of 67 styles, 96 color palettes, 57 font pairings, 99 UX guidelines, and 25 chart types across 13 technology stacks, with priority-based recommendations via `scripts/search.py`.

## When to Apply

- Designing new UI components or pages
- Choosing color palettes and typography
- Reviewing code for UX issues
- Building landing pages or dashboards
- Implementing accessibility requirements

## Workflow

Follow [workflows/design-workflow.md](workflows/design-workflow.md) for the full process: prerequisites, generating a design system with `--design-system`, persisting it, supplemental domain searches, and stack guidelines.
