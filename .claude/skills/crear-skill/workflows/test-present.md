# Pasos 6–7: Testear y Presentar

## Paso 6 — Testear

Después de instalar:

1. Simulo ser usuario nuevo
2. Pruebo que las instrucciones son claras
3. Si genera archivos, verifico que funcionan
4. Ajusto si algo no fluye

Checklist:
- [ ] Las instrucciones son claras
- [ ] El flujo es conversacional (no un formulario)
- [ ] Los ejemplos son útiles
- [ ] Si genera archivos, existen y funcionan
- [ ] Puedo entenderla en 30 segundos

## Paso 7 — Presentar al Usuario

Muestro:

1. **Nombre y ruta:**
   ```
   Skill creada: my-skill
   Ubicación: .claude/skills/my-skill/SKILL.md
   ```

2. **Frases que la activan:**
   ```
   "crear propuesta", "generar presupuesto", 
   "proposal generator", etc.
   ```

3. **Qué hace:**
   ```
   Input: Datos del cliente
   Output: Propuesta en PDF/HTML
   ```

4. **Cómo usarla:**
   ```
   /my-skill
   [responde a las preguntas]
   ↓
   Propuesta generada en archivo
   ```

5. **¿Quieres ajustar?**
   ```
   ¿Hay algo que cambiar?
   - Agregar más campos
   - Cambiar el formato
   - Renombrar
   ```

**NO incluyo:** Precios sugeridos, "como servicio", consejos de venta.
