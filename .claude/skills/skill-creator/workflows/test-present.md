# Steps 6–7: Test & Present

## Step 6 — Test

After installing:

1. Act as a new user
2. Verify instructions are clear
3. If it generates files, check they work
4. Adjust if anything doesn't flow

Checklist:
- [ ] Instructions are clear
- [ ] Flow is conversational (not a form)
- [ ] Examples are helpful
- [ ] If it generates files, they exist and work
- [ ] I can understand it in 30 seconds

## Step 7 — Present to User

Show:

1. **Name and location:**
   ```
   Skill created: my-skill
   Location: .claude/skills/my-skill/SKILL.md
   ```

2. **Activation phrases:**
   ```
   "create proposal", "generate budget", 
   "proposal generator", etc.
   ```

3. **What it does:**
   ```
   Input: Client data
   Output: Proposal in PDF/HTML
   ```

4. **How to use it:**
   ```
   /my-skill
   [answer the questions]
   ↓
   Proposal generated in file
   ```

5. **Want to adjust?**
   ```
   Anything to change?
   - Add more fields
   - Change format
   - Rename it
   ```

**DO NOT include:** Suggested pricing, "as a service", sales advice.
