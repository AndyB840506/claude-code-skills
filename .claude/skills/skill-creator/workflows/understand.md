# Step 1 — Understand What You Need

Ask conversationally (not as a form).

## Key Questions

- **What language?** — Spanish or English
- **What do you want to automate?** — Describe the process or outcome
- **What information does it need?** — URL, text, folder, data, file...
- **What should it generate?** — HTML, report, code, file, dashboard...
- **Who uses it?** — Just you, your team, or publicly?

## If You're Unsure

Suggest ideas by industry:

**For Business:**
- Proposal generator (client data → PDF/HTML)
- Budget calculator (hours/services → quote)
- Contract generator (data → personalized contract)
- Sales presentation creator

**For Marketing:**
- Ad copy generator (product → variants)
- Content planner (niche → 30-day calendar)
- Sales email creator (product → sequences)
- Social media post generator (topic → IG/LinkedIn/X)

**For Development:**
- API generator (model → complete API)
- Code documenter (repo → docs)
- Test generator (code → test suite)
- Project scaffolder (type → structure)

**For Productivity:**
- Document summarizer (PDF → summary)
- Meeting transcriber (notes → minutes)
- SOP generator (process → procedure doc)
- Data analyzer (CSV → dashboard)

## If They've Described Enough

Proceed directly to Step 2 (Design) without more questions.
