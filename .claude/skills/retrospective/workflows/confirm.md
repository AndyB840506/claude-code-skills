# Step 4: Ask for Confirmation

**CRITICAL:** Before applying ANY changes, explicitly ask:

```
¿Quieres que aplique estos cambios?

Cambios propuestos:
1. .claude/skills/X — Add learning X
2. .claude/skills/Y — Add learning Y

- "sí" / "yes" — Aplicar todos
- "solo X" — Solo cambios #X
- "no" / "skip" — No aplicar nada
```

**Wait for explicit user confirmation.** Do NOT proceed without approval.

This ensures user has full control over skill modifications.
