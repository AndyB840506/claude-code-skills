# Screening Report Template

Use this structure for all candidate reports. File saved as `report-[candidate-name]-[YYYY-MM-DD].md`

---

```markdown
# Screening Report

**Candidate:** [Name]  
**Position:** [Job Title]  
**Date:** [YYYY-MM-DD]  
**Interviewer:** Smart Recruiter Skill

---

## Verdict: [Strong Fit / Possible Fit / Not a Fit]

**Overall Score: [X.X / 10]**

| Dimension | Score |
|---|---|
| Must-have requirements | X/10 |
| Cultural fit | X/10 |
| Motivation & expectations | X/10 |
| Communication & soft skills | X/10 |

---

## Strengths

- [Strength 1 — with specific evidence from interview]
- [Strength 2 — with specific evidence]
- [Strength 3 if applicable]

## Concerns / Gaps

- [Concern 1 with specifics]
- [Concern 2 with specifics]

## Notable Quotes

> "[Key quote supporting assessment]"

> "[Another relevant quote if applicable]"

## Salary Expectations

[What the candidate said about salary / "Not discussed"]

## Recommendation

[2–3 sentence recommendation for the hiring team. Be specific about next steps.]

---

*Generated by Smart Recruiter Skill*
```

## Notes

- All information should be evidence-based from the actual interview
- Never invent or assume details
- Quotes should be paraphrased or exact (put in quotes if exact)
- Recommendation should be actionable for the hiring team
